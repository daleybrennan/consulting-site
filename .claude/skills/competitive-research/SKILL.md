# Competitive-set research

Find the wines a given wine would actually sit beside in a target market, with real prices, so the pricing walk can be judged against the shelf.

## Inputs you need
Wine style/grape, origin, quality tier, target market (+ region/state), and the client's stated positioning ambition / comparables they admire.

## Method
Use web search. Build a set of **6–12 genuinely comparable** wines — same style, similar quality tier and origin reputation, actually present in the target market. For each, capture:
- Name, producer, vintage if shown
- Style / origin
- **Off-premise (retail) price** — from retailer listings, Wine-Searcher public pages, importer/distributor catalogues
- **On-premise (list) price** — from restaurant wine lists where findable
- Importer / distributor if visible
- The source URL and the date observed

Prefer the **specific target region** (e.g. New York retailers for NY) over national averages — pricing is local.

## Output
A table-ready list plus:
- The observed **price band** (low–high) for the set, off-premise and on-premise
- Where the modelled wine would land **within** that band
- 2–3 sentences on the shape of the set (who owns the placement, where it's crowded vs. open)

## Rigour (this feeds a $5,000 document)
- Cite every price with its source and date; flag anything you couldn't verify as an estimate.
- Distinguish observed prices from inferred ones.
- Don't invent comparables or prices. If the set is thin, say so and widen the net deliberately (adjacent regions/styles), noting the adjustment.
- Hand the sources to the orchestrator for the appendix and for Daley's verification before sign-off.
- Prose describing the competitive set must be professional and precise — no informal evaluative language ("a prestige scalp", "a steal") and no trade argot.
